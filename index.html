Objetivo

Esse algoritmo tem como objetivo encontrar a solução com o menor custo para as matrizes em questão:

{% include collapse-matrizes.html%}

Regras

Para resolver a matriz, algumas regras devem ser obedecidas:

  • Escolha uma sequência, tendo a fábrica como ponto de partida e chegada, exemplo: Fábrica>P1>P2>P3>..PN..>Fábrica
  • Não Repita Números

Solução

Após selecinada a sequência, deve-se calcular o custo do ponto atual para o próximo, por exemplo:

    Sequência [Fábrica(F),P3,P2,P1,P4,Fábrica(F)] na primeira matriz:

  • De: F Para: P3 = 13;
  • De: P3 Para: P2 = 9;
  • De: P2 Para: P1 = 9;
  • De: P1 Para: P4 = 18;
  • De: P4 Para: F = 6;
No fim, soma-se o custo de cada operação e verifica-se qual o custo total produtivo (55).
